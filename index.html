GitHub es más que un simple repo, es una red social con casi todos los mecanismos que tiene una de ellas: "like" (en forma de estrellas, watches y forks), "muro" en forma de issues, que sirven para todo tipo de interacción, pero lo más interesante es que, a base de forks y pull requests, se crea una verdadera comunidad de usuarios. Todo esto se refuerza con las referencias desde los commits, que es otro mecanismo de interacción. El "muro" de GitHub es un verdadero muro de red social. Y como LinkedIn, es una red social que te puede conseguir un trabajo

Mira la ruleta del push y los duelos a primer commit
